# .metaHub

Self-contained hub for all meta infrastructure, configurations, and organizational resources.

## Structure

```
.metaHub/
├── config/       # All configuration (Claude, VSCode, DevContainer, etc.)
├── governance/   # Governance structures and policies
├── archives/     # Historical archives
├── tools/        # Automation, dev-tools, git hooks
├── knowledge/    # Knowledge base
├── docs/         # Documentation
├── scripts/      # Management scripts
└── templates/    # Reusable templates
```

## Version

Current version: v1.0.0

## Note

The `.organizations` directory remains separate outside .metaHub as per organizational requirements.

See individual subdirectories for detailed documentation.
