## Usage Tips

### Finding Documentation in Cloned Repos

Most cloned repos have docs in:
- `docs/` - Main documentation folder
- `README.md` - Overview and quickstart
- `examples/` - Usage examples
- `CONTRIBUTING.md` - How it works internally

### Converting to Searchable Format

```bash
# Convert all markdown to searchable index
find . -name "*.md" -exec cat {} \; > all-docs.txt

# Create a grep-able docs archive
find . -name "*.md" | xargs grep -l "tutorial\|example\|guide" > guides.txt
```

### Using with AI Tools

Feed these docs to your AI assistant for context:

```bash
# Example: Give Aider context about Continue.dev
aider --read 01-major-assistants/continue-dev/docs/

# Example: Search all docs for a pattern
grep -r "MCP\|model context protocol" --include="*.md"
```

## Updating Documentation

```powershell
# Update all cloned repos
Get-ChildItem -Recurse -Directory -Filter ".git" | ForEach-Object {
    Push-Location $_.Parent.FullName
    git pull --quiet
    Pop-Location
}
```
