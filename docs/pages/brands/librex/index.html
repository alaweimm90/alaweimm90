Optimization Framework

Librex

A unified optimization framework with modular solvers for complex combinatorial and continuous problems. From Latin "libra" (balance) + "rex" (king).

Solvers

Librex.QAP

Quadratic Assignment Problem solver for facility layout and network design.

pip install librex-qap

Librex.Flow

Network flow optimization for supply chains and logistics.

pip install librex-flow

Librex.Alloc

Resource allocation for workforce and capacity planning.

pip install librex-alloc

Librex.Evo

Evolutionary optimization with genetic algorithms.

pip install librex-evo

Librex.Graph

Graph optimization for routing and network problems.

pip install librex-graph

Librex.Dual

Dual decomposition for large-scale problems.

pip install librex-dual

Librex.Meta

Meta-optimization for automatic algorithm selection.

pip install librex-meta
