## üõ†Ô∏è How to Run

### Prerequisites
- Windows PowerShell (or PowerShell Core)
- Node.js 20+
- Docker Desktop
- Git

### Quick Start
```powershell
# Run the complete 500-step workflow
.\scripts\BENCHBARRIER_AUTONOMOUS_WORKFLOW.ps1

# Run specific step range
.\scripts\BENCHBARRIER_AUTONOMOUS_WORKFLOW.ps1 -StartStep 1 -EndStep 100

# Skip tests but force deployment
.\scripts\BENCHBARRIER_AUTONOMOUS_WORKFLOW.ps1 -SkipTests -ForceDeploy
```

### Parameters
- `-StartStep`: Starting step number (default: 1)
- `-EndStep`: Ending step number (default: 500)
- `-SkipTests`: Skip automated testing phases
- `-ForceDeploy`: Force deployment even with errors
